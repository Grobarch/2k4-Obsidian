# Templates

Szablony Obsidian — niewidoczne w widoku web (folder ignorowany przez Quartz).

## Dostępne szablony

- **Utwórz Postać** — formularz tworzenia bohatera gracza lub niezależnego
  - Wymaga: plugin **Templater** + plugin **Meta Bind**
  - Uruchamiany przyciskiem z folderu kampanii lub encyklopedii

## Zarządzanie obrazami

Plik `Zarządzanie obrazami.md` opisuje jak dodawać pliki `.jpg`, `.png`, `.svg` do notatek:
- obsługiwane formaty i składnia osadzania
- gdzie umieszczać pliki graficzne (obok notatki lub w `assets/`)
- zachowanie w widoku web (niewidoczne w Explorer i wyszukiwarce)

## Statbloki systemów

Folder `statblocks/` zawiera statbloki dla poszczególnych systemów RPG.
Każdy plik jest wklejany do tworzonej notatki postaci.
